# Wobe Website

Jekyll based website for marketing purposes.

## Development

Follow these steps:

1. `gem install`
2. `jekyll serve`

## Deployment

To deploy the website, you need to have the right variables in the
`.env` file.

1. `cp env.sample .env`
1. `rake deploy`

After deploy, the script will print the url of the live website.
